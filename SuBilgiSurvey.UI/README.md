# SuBilgiSurvey.UI

Next.js (App Router) + MUI + TanStack Query + Zustand.

```bash
cp .env.local.example .env.local
npm install
npm run dev
```

- Admin: `/admin/*` (JWT rolü `Admin`)
- Kullanıcı: `/my-surveys`
- Giriş: `/login`, kayıt: `/register`

API adresi: `NEXT_PUBLIC_API_URL` (varsayılan http://localhost:5083).
